## Links

## Tech-stack

Front-end:

- [Soild.js](https://www.solidjs.com/)
- TypeScript
- Bootstrap
- [Vite](https://vitejs.dev/)
- [PNPM](https://pnpm.io/)

This web app uses local-storage to store and retrieve saved repositories.
